# 🛒 Superstore Sales Dashboard - Power BI

## 📌 Objective
To contribute to the success of a supermarket business by utilizing data analysis 
techniques, specifically focusing on time series analysis, to provide valuable 
insights and accurate sales forecasting.

## 📊 Dashboard Overview
An interactive Power BI dashboard designed to help business stakeholders 
make data-driven decisions through:

- ✅ KPI tracking and monitoring
- ✅ Interactive visualizations with filters
- ✅ Sales trends across multiple dimensions
- ✅ sales forecasting using time series analysis

## 🔍 Key Components

| Feature | Description |
|--------|-------------|
| Dashboard Creation | Intuitive visuals with drill-down filtering |
| Data Analysis | Insights on sales strategy effectiveness |
| Sales Forecasting |  forecast using historical data |
| Actionable Insights | Recommendations to drive growth & efficiency |

## 🛠️ Tools Used
- **Power BI** – Dashboard creation & visualization
- **Excel** – Data cleaning and preparation

## 📈 Key Insights
- Identified top-performing product categories
- Tracked monthly revenue trends
- Forecasted next sales using time series
